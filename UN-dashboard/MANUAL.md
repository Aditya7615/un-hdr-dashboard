# UN HDR Dashboard — User Manual

## Overview

This interactive dashboard visualizes data extracted from the **2016 UN National Human Development Report for Serbia**, which examines the role of **social capital in community resilience** to natural disasters. The dashboard has **6 tabs**, each providing a different analytical view.

---

## 1. Thematic Density Tab

Shows keyword frequency distribution across 7 themes extracted from the report.

**Controls:**
- **Filter themes** — Multi-select to show/hide specific themes
- **Min %** — Minimum density threshold (0–50%)
- **Type** — Switch between Bar, Pie, or Treemap charts

**Use case:** Quickly identify which thematic areas (Climate, Economy, Gender, etc.) receive the most attention in the report.

---

## 2. Cross-Model Validation Tab

Compares extracted numerical indicators against simulated alternative extractions to assess reliability.

**Controls:**
- **Indicators** — Select which indicators to compare
- **Model variant** — Choose "Extracted Value", "Simulated Run 2", or "Simulated Run 3"
- **Mode** — "Grouped Bar" for side-by-side comparison or "Scatter" with trendline

**Use case:** Evaluate the stability of extracted values. Tight clustering = high confidence.

---

## 3. Time Trends Tab

Plots historical data extracted from the report (flood impacts, HDI progression, economic damage, temperature anomalies).

**Controls:**
- **Indicators** — Select one or more indicators to overlay
- **Year range** — Slide to focus on specific time periods
- **Type** — Line, Area, or Bar chart
- **Log scale** — Toggle logarithmic Y-axis (useful when comparing indicators with very different scales)

**Use case:** Track how flood impacts and human development have changed over time in Serbia.

---

## 4. Resilience Capitals Tab

Radar chart of the **5 Capitals of Resilience**: Natural, Physical, Economic, Human, and Social/Institutional.

**Controls:**
- **Search municipality** — Filter the entity ranking table
- **Compare** — Switch between "Single" (baseline only) and "Multi-select" (overlay municipality comparisons)
- **Show top N** — Control how many municipalities appear in the ranking bar chart

**Interpretation:**
- The blue radar shows the Serbia average for each capital
- Multi-select overlays scale each capital by the municipality's I-distance score
- A larger radar area = higher resilience

---

## 5. Correlation Matrix Tab

Pearson correlation heatmap of indicators from the I-distance analysis, plus a histogram of normalized I-distance scores.

**Controls:**
- View the full correlation matrix (numerical values shown in each cell)
- Scroll down for the I-distance score distribution

**Use case:** Identify which indicators move together (positive correlation) or inversely (negative correlation).

---

## 6. Country Explorer Tab

Browse **117 student country assignments** extracted from the assignment PDF.

**Controls:**
- **Search** — Find by student name, country, or ID number
- **Filter by country** — Multi-select to show specific countries only
- **Download CSV** — Export the filtered table

**Use case:** Quickly look up which country a student was assigned to research.

---

## FAQ

**Q: The dashboard shows "No data" for a tab.**
A: Click the **Reload data** button in the sidebar. If the issue persists, run `python pipeline.py --mock` to regenerate the JSON output files.

**Q: Can I use my own LLM for extraction?**
A: Yes. Create a `.env` file with `GROQ_API_KEY=your_key_here` and run `python pipeline.py` without `--mock`.

**Q: The time series chart is empty.**
A: The dashboard uses built-in fallback data. Run the full pipeline with an LLM key to extract richer time series from the report PDF.

**Q: How do I update the country assignments?**
A: Replace `data/assignment_1_country_assignment.pdf` with your own PDF and run `python pdf_processor.py`.
